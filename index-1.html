Slider 2D

This tool is meant to provide a two dimensional slider that allows you to make more intricate input forms.

It does this by always returning a value between 0% and 100% on either the x or y axis for each handle and mapping it to your provided number range, thus having a completely resizable input element without any rounding or pixel size errors.

Additionally, different handles can be linked, allowing their bounds to be modified by the linked handles' position.
